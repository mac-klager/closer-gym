# Transcript Calibration

Drop real sales call transcripts here as `.txt` or `.md` files.

On every new call, excerpts from these files are injected into both the
persona generator and the prospect roleplay prompt as **voice calibration** —
the AI studies how real prospects in your niche hesitate, deflect, and slowly
reveal the truth, and matches that realism.

Tips:
- Plain text works best. Format doesn't matter much; speaker labels help
  (e.g. `Closer:` / `Prospect:`).
- The first ~12,000 characters across all files are used, so put your best
  transcript first (files are read alphabetically — prefix with `01-`, `02-`...).
- To convert a PDF transcript to text on macOS:
  `mdimport` won't do it, but this will if you have it: open the PDF, select all, paste into a `.txt` file —
  or ask Claude Code to extract it for you.
